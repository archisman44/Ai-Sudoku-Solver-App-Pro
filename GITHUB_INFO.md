## Description
AI-powered Sudoku Solver with multiple algorithms, image recognition, and step-by-step visualization. Built with React, Flask, OpenCV, and advanced solving algorithms.

## Tech Stack
- **Frontend:** React 18, JavaScript, CSS3
- **Backend:** Python 3.10+, Flask, OpenCV
- **AI/CV:** Tesseract OCR, OpenCV
- **Algorithms:** Backtracking, Constraint Propagation, Dancing Links, Heuristic Solver

## Features
✅ Multiple solving algorithms  
✅ Image recognition & OCR  
✅ Puzzle generation (5 difficulty levels)  
✅ Step-by-step visualization  
✅ Hint system  
✅ Dark/Light mode  
✅ Real-time validation  
✅ Performance statistics

## Local Setup
```bash
# Backend
cd backend
pip install -r requirements.txt
python app.py

# Frontend
cd frontend
npm install
npm start
```

Access at http://localhost:3000
